Tmac - MAC changer for Windows

To change MAC in macOS: 

$ sudo ifconfig  enX ether [ne:wm:ac:ad:dr:es:s0]

(need to be su ofc)

To change MAC in Linux: 

$ sudo macchanger -r ethX

(-r for random, need to down adapter before change, need to be su ofc!)
